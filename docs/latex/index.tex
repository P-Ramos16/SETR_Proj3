Repository for the third project of the 24-\/25 SETR class at Universidade de Aveiro A Zephyr RTOS-\/based temperature regulation system with PID control, UART command interface, and LED status indicators.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Real-\/time temperature monitoring via I2C (TC74 sensor)
\item PID controller for precise temperature regulation
\item Heater control via FET
\item UART command interface for system control
\item LED status indicators
\item Button controls for manual operation
\item Thread-\/safe RTDB (Real-\/\+Time Database) for data sharing
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Thread Architecture}{Thread Architecture}}\label{index_autotoc_md2}
\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Threads}{Threads}}\label{index_autotoc_md3}

\begin{DoxyEnumerate}
\item {\bfseries{LED Update Task}}\+: Updates LED indicators based on system state
\item {\bfseries{Temperature Reading Task}}\+: Reads temperature from sensor
\item {\bfseries{PID Controller Task}}\+: Calculates PID output
\item {\bfseries{Heat Control Task}}\+: Controls heater based on PID output
\item {\bfseries{UART Command Task}}\+: Processes incoming UART commands
\end{DoxyEnumerate}\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{UART Commands}{UART Commands}}\label{index_autotoc_md4}
