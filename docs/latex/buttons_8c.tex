\doxysubsection{Detailed Description}
Gestão dos botões físicos do sistema. 

Configura os botões BTN1, BTN2 e BTN4 para operar por interrupção. Cada botão invoca um callback\+:
\begin{DoxyItemize}
\item BTN1\+: Liga/desliga o sistema.
\item BTN2\+: Aumenta a temperatura desejada.
\item BTN4\+: Diminui a temperatura desejada.
\end{DoxyItemize}

Cada ação manipula a base de dados em tempo real (RTDB), assegurando a sincronização entre tarefas. \begin{DoxyAuthor}{Author}
