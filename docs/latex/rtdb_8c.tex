\doxysubsection{Detailed Description}
Real-\/\+Time Database (RTDB) para sincronização entre tarefas. 

Armazena variáveis partilhadas do sistema como\+:
\begin{DoxyItemize}
\item Estado ON/\+OFF do sistema
\item Temperatura desejada
\item Temperatura atual (medida)
\end{DoxyItemize}

Todos os acessos são protegidos por {\ttfamily k\+\_\+mutex}, garantindo integridade em ambiente multitarefa. Fornece funções {\ttfamily get} e {\ttfamily set} para abstrair o acesso concorrente aos dados.
